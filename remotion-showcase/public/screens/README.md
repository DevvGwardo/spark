# Spark screenshots for the showcase video

Capture these from the **running Spark Electron app in dark mode**, then save each
into this folder using the **exact filename** shown. The video references these names
directly — drop your real captures in (overwriting the placeholders) and re-render.

> Tip: macOS `Cmd+Shift+4` then **press Space** to capture a single window cleanly
> (with shadow), or drag to select a region. Captures are Retina/2× automatically —
> perfect, the higher-res the better.

| Filename             | What to capture                                                                                 |
|----------------------|-------------------------------------------------------------------------------------------------|
| `01-app-full.png`    | The **whole Spark window** — a chat open in the main area with the Hermes sidebar visible. Hero shot. |
| `02-overview.png`    | The **Hermes Overview** sidebar panel (Tracked Sessions / Live Sessions / Cron Jobs / Skills).  |
| `03-usage.png`       | The **Hermes Usage** panel, top portion — the Sessions/Messages/Input/Output cards + Cost + Total Tokens + Top Models bars. |
| `04-charts.png`      | The **Hermes Usage** panel scrolled to the charts — Recent Activity histogram + Token Usage Over Time area chart. |
| `05-sessions.png`    | The **Hermes Sessions** panel — the list of tracked sessions with status/message counts.        |
| `06-chat.png`        | An **open chat transcript** (a session replay or a live conversation with the agent).           |

Any aspect ratio is fine — the video frames each shot with `object-fit: contain`,
so nothing gets cropped awkwardly. Sidebar panels (tall & narrow) look great in this
vertical 9:16 format.
