# Spark Brand Notes

## Positioning

Spark is the desktop AI workspace built around Hermes: chat when you want it, deploy an autonomous agent when you need it.

## Tagline

Primary tagline:

`Chat with AI. Deploy an agent. Spark something.`

Backup tagline:

`The AI desktop with an autonomous agent brain.`

## Voice

- Bold, technical, fast.
- Dense with signal, never cute.
- Agent-first without sounding magical or vague.

## Visual Direction

- Dark-native interface with restrained warm orange accents.
- Flame-and-spark mark to suggest ignition, momentum, and autonomous action.
- Crisp typography and developer-tool polish over decorative gradients.
