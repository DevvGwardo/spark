settings — providers
One place for every key

Bring your own models. All of them.

Spark works as a clean chat client with any provider — Hermes is optional. Manage every key, sign in with OpenRouter OAuth, and switch the active model without leaving the conversation.

  • OAuth or API key — drop in a key or sign in, your choice.
  • Instant model switch — change models mid-thread, no reload.
  • Messaging & GitHub integrations — wired into the same settings surface.
⌘K — command palette
Keyboard-first, no friction

Jump anywhere without touching the mouse.

Hit ⌘K to fly between conversations, profiles, and actions. The whole app is built around staying in flow — find a thread, switch a model, fire an action, and get back to the code.

  • Tags & archive — color-coded chips and a sidebar filter keep the list clean.
  • Import / export — conversations as JSON or Markdown.
  • Change Approval modal — approve risky tool calls once, for the session, or always.
